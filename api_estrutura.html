ENTENDA COMO NOSSA API FUNCIONA

O Webservice do coronavirus foi criado gratuitamente para que a populacão possa a ter acesso dos dados atraves de aplicações desenvolvidas por programadores ou por pessoas de conhecimentos necessarios. Pelo fato da facto de ser gratuito pedimos a colaboração de todos informando " casos suspeitos", contamos com a sua ajuda. Clique aqui para baixar a aplicação.

A estrutura da nossa api possui dois metodos sendo eles o "Listar" e o "Encontrar" , como o proprio nome sugere o listar retorna uma lista de todos as provincias em ordem crescente envolvendo os numeros de casos registrados, ja o encontrar necessita de um parâmetro, pois trata de um mêtodo de pesquisa, ou seja, o retorno depende do parametro passado via URL. Para ambos os casos o resultado sera uma lista de dados retornado via json.

ACESSANDO O WEBSERVICE

Os dados que alimentam a api são colhidos no site do ministerio da saúde, através de grandes portais de noticia, plataformas de monitoramento especializada. Tantos os dados como a estrutura da api pode ser modificada ao longo do tempo, sendo assim pedimos a vossa compreensão e colaboração. Mas contamos com a sua informação sobre possiveis casos.
